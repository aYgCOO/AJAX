jQuery Syntax and Concepts

Basic jQuery Syntax and Concepts

1. Including jQuery

To use jQuery, include it in your HTML file. You can do this by linking to a CDN or downloading it and including it locally.

<!-- CDN -->
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>

2. Document Ready Event

Ensures that the DOM is fully loaded before executing any jQuery code.

$(document).ready(function(){
    // Your code here
});

Or the shorthand:

$(function(){
    // Your code here
});

3. Selecting Elements

jQuery uses CSS selectors to select HTML elements.

// Select elements by tag
$("p").hide();

// Select elements by class
$(".myClass").hide();

// Select elements by ID
$("#myId").hide();

4. Event Handling

Attach event handlers to elements.

// Click event
$("button").click(function(){
    alert("Button clicked!");
});

// Hover event
$("p").hover(function(){
    $(this).css("color", "red");
}, function(){
    $(this).css("color", "black");
});

5. Manipulating CSS

Change the style of elements.

$("p").css("color", "blue");
$("p").css({"color": "blue", "font-size": "20px"});

6. Hiding and Showing Elements

Show or hide elements with effects.

$("p").hide();
$("p").show();

7. Adding and Removing Classes

$("p").addClass("myClass");
$("p").removeClass("myClass");

8. Getting and Setting Content

Get or set the HTML or text content of elements.

// Get content
var htmlContent = $("#myDiv").html();
var textContent = $("#myDiv").text();

// Set content
$("#myDiv").html("<p>New HTML content</p>");
$("#myDiv").text("New text content");

Intermediate jQuery Syntax and Concepts

1. Chaining

Perform multiple actions on the same set of elements.

$("#myDiv").css("color", "red").slideUp(2000).slideDown(2000);

2. Effects and Animations

Create animations and effects.

// Fade out
$("#myDiv").fadeOut("slow");

// Slide up
$("#myDiv").slideUp("slow");

// Custom animation
$("#myDiv").animate({
    opacity: 0.5,
    height: "50%"
}, 2000);

3. Traversing the DOM

Navigate through the DOM tree.

// Find all <span> elements inside <div>
$("div").find("span").css("color", "red");

// Get parent element
$("span").parent().css("color", "blue");

// Get children elements
$("div").children().css("color", "green");

// Siblings
$("p").siblings().css("color", "yellow");

Advanced jQuery Syntax and Concepts

1. Creating Custom Plugins

Extend jQuery with your own functions.

$.fn.myPlugin = function() {
    this.css("color", "red");
    return this;
};

// Usage
$("p").myPlugin();

2. Deferred and Promises

Handle asynchronous operations more effectively.

var deferred = $.Deferred();

deferred.done(function(){
    console.log("Task completed");
});

deferred.resolve();

4. Data Methods

Work with HTML5 data attributes.

// Set data
$("div").data("key", "value");

// Get data
var value = $("div").data("key");

5. Event Delegation

Efficiently handle events for dynamic content.

// Event delegation
$(document).on("click", "button", function(){
    alert("Button clicked!");
});
